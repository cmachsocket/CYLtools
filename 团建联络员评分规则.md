# 团建联络员评分规则（由 `团建联络员打分表.csv` 提取）

## 1. 评分维度
- 人员出勤
- 流程完整度
- 成员参与度
- 主题相关度
- 联系团建联络员情况

## 2. 总体说明
- 所有打分均按 100 分为满分赋分，比例已后期折合。

## 3. 各维度规则
### 3.1 人员出勤
- 按出勤率直接打分，如 100% 出勤为 100 分，91% 出勤为 91 分。
- 出勤率 = 到场人数 / 班级人数。

### 3.2 流程完整度
- 重点关注是否有唱团歌环节（必要环节）。
- 团支部是否围绕每月相关主题开展主题团日活动。
- 不包括班会、班委会等其他形式的班集体活动。
- 不能出现将团日活动并入其他班级活动一起开办的情况。
- 团支书是否按要求认真撰写主题团日活动总结表和新闻稿（必要环节）。
- 其余由团建联络员根据活动开展流畅度、活动内容完成情况酌情给分。

### 3.3 成员参与度
- 关注是否有除主讲人外同学参与研讨。
- 参与形式包括但不限于：发表感想、提出问题、回答问题、参与讲述团日活动内容等。
- 由团建联络员视情况给分。

### 3.4 主题相关度
- 主题团日活动是否与当月建议主题相关（基本分 85 分）。
- 若在当月主题基础上有所延申，视情况提高分数。

### 3.5 联系团建联络员情况
- 若主动邀请团建联络员参加团日活动：满分。
- 若活动开展后主动告知或发送材料：50 分。
- 若直到团建联络员提醒才告知已开展：0 分。

## 4. 附加说明（注）
1. 所有团日活动均需打分，无论是否现场参与。
2. 现场参与按现场情况打分；未现场参与按接收材料打分；未发材料按 0 分处理。
3. 允许多班联办，但以上团支部不允许提交相同材料。
4. 若材料相同，视为未认真撰写材料，在“流程完整度”扣 20 分。
5. 上交材料中需写清各团支部人员出勤率。
6. 对硬性要求部分（出勤、团歌、材料提交）按上交材料打分时：
   - 出勤率需提供一张能覆盖所有人的照片；否则按人数最多的一张照片计算。
   - 团歌需有视频材料证明。
7. 根据材料打分时，通过照片、视频及新闻稿、总结表等文字材料酌情给分。
8. 若有团支部截止后仍未开展团日活动，团建联络员需将相关团支部编号和所属学院告知各组负责人。
